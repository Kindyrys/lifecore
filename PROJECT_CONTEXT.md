# LifeCore — Project Context

## Project name

LifeCore

## Goal

LifeCore is a Java Spring Boot pet project created to learn backend development and build a portfolio project for Junior Java Developer positions.

The application will help users manage:

- personal finances;
- tasks;
- habits;
- daily and monthly dashboard overview.

## Learning goal

The main goal is not only to build the application, but to learn how to:

- understand Java backend architecture;
- write clean Java code;
- use Spring Boot;
- work with PostgreSQL;
- structure a real backend project;
- explain code during a Junior Java interview;
- work with Git and GitHub professionally.

## Tech stack

Current stack:

- Java 17
- Spring Boot
- Maven
- PostgreSQL
- Spring Data JPA
- Spring Security
- Thymeleaf
- Bootstrap
- Git
- GitHub
